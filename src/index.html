# Algorithm Description
1 Prim Find a minimum spanning tree in weighted graph through local search.
2 Kruskal Find a minimum spanning tree in weighted graph.
3 Breadth-First Search Search algorithm that prioritarily explores nodes neighbors as they are put in a queue.
4 Depth-First Search Search algorithm that prioritarily explores nodes neighbors as they are put in a stack.
5 Dijkstra's Algorithm Find a shortest-path tree in a weighted graph.
6 Welch e Powell Graph node coloring algorithm for colision problems.
